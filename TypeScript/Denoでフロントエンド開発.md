# Node.jsと比較したDeno
- npm依存にマルウェアが混入した場合への防衛
- TypeScript型チェック、Lint、Formatterが付属  
- Node.jsを100%互換しているわけではない
- **新規プロジェクトでDenoを採用するのがおすすめだが、Next.jsは厳しい**

# ひとこと
